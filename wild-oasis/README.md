Next Wild Oasis Final Project for React / Nextjs Course

Description:
Next Wild Oasis is a booking applications for the Next Wild Oasis hotel. It allows the staff to enter rooms,

This is my final project for the React / Nextjs course. The original project was to be split into 2 version a React version and a Nextjs version using API routing. I decided to have the inital project be solely Nextjs using the clean arctecture, Nextjs actions instead of React Query for caching, Prisma ORM is used as a layer instead of directly connecting to Supabase (I wanted something local that was easily updatable and could be pointed to another db if needed), zod for validation through the client and server, Tailwind where it was applicable but there is raw CSS for a number of components and redesign of a number the components (my own take on them).

Using my background and experience I was able to develope my own vision of the final project which implementated all the nessary requirements for the use case(s). I thought it was important to do this in order to differient myself of the other projects and showcase my skill set in my profolio.

pnpm was used a package manager
